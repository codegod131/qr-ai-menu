# Frontend Actionable Tasks
**Owner:** Member 3 (Role C)
**Stack:** Next.js (or Vite+React), Tailwind CSS

## Day 1: Foundations
- [ ] Initialize the Next.js (or Vite) project with Tailwind CSS.
- [ ] Build the Client Portal skeleton (Web Portal):
  - Simple login form requesting Business Code.
  - Add-item form (Name, Price, Description, Tags, optional Photo).
- [ ] Build the Customer Page skeleton (Catalog Page):
  - Search input box for natural language queries.
- [ ] Build QR code generation component (`qrcode.react`) pointing to a static/dummy business URL.

## Day 2: Core Loop
- [ ] Connect frontend to live backend APIs (Item creation, Querying).
- [ ] Update QR generation to encode the real business URL.
- [ ] Build search results UI showing AI explanation ("why this matches") per result.
- [ ] Implement quick thumbs up/down buttons on each search result for feedback.

## Day 3: Measure + Learn + Polish
- [ ] Mobile polish: ensure the UI is phone-first, responsive, and easy to tap on an actual phone.
- [ ] Add robust loading states (skeletons/spinners) and empty states.
- [ ] Fix obvious bugs discovered during full team QA.
